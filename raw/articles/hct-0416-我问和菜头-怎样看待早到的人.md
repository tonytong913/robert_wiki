# 我问和菜头：怎样看待早到的人（hct-0416）

> 来源：和菜头《成年人修炼手册》第二季 Q&A
> 日期：2026-04-16

## 全文

本期为 Q&A 合集，涵盖以下主题的问答：

### 一、关于"胜负：执着于胜负的人会失去更多"

**用户纳博（育儿问题）：**
儿子初一，说什么都不想争，把成绩跌到谷底自嘲。热爱写作，疫情在家写了30多万字小说。但数学、体育一遇困难就放弃，现在甚至不想面对数学。怎么鼓励他重建面对困难的勇气？

**和菜头回答要点：**
- 不应该找我，应该求教于资深教师
- 自己经历：不是靠勇气，而是**努力找寻最小的成就感**
- 策略：不把时间花在从90分提到95分，而是把另一门从45分提到60分——15分比5分好赚
- 对90分和60分**同样得意**
- **最重要**：不会因为有一门课可以轻易获得成就感就反复投入其中——不反复只获取那一点成就感

### 二、关于"负面情绪比快乐靠得住"

**用户姚涛（职业转型勇气）：**
几年前想换行业，犹豫彷徨痛苦，像站在悬崖边想奋力一跃但怕粉身碎骨。最后留在现在行业，虽发展还可以但总在想"如果当时跳了会怎样"。如何增加一点勇气？

**和菜头回答要点：**
- 无论跳不跳，差别可能都不大——因为你有适应能力
- 那些要跳的人，早就因为适应不了而纵身跳了下去
- 现在的不甘心是**妄念**：认为现在做得好，当初换条路会更好
- 既然做出了选择，不妨**天真地相信自己的选择就是最好的**
- "差一点"就是人生里的十万光年——不存在差一点就怎样的说法

**用户王志文（精神痛苦 vs 身体痛苦）：**
负面情绪是精神层面的产物，但为什么来袭时会胸闷发慌，像大石头压在胸口——到底是精神痛苦还是身体痛苦？

**和菜头回答要点：**
- 精神痛苦当然会带来肉体痛苦，这是**常识**
- 精神紧张→肠应激症→想跑厕所；压力过高→恶心反胃→反射式呕吐；惊吓刺激→心跳过速血压上升→可能晕厥
- 都没有任何器质性损害——都是心理变化影响生理
- 感觉胸口有大石、心慌气闷=情绪刺激过于强烈，达到身体承受极限
- 应对：深呼吸增加供氧、适当起身运动分散注意力
- 好消息：人的适应力很强，类似状况会逐渐减弱

### 三、关于"守时帮你找到自己人"

**用户xx（不守时保持若即若离）：**
既然不守时表明不愿融入圈子，为什么还要用不守时来保持若即若离？不去不就好了吗？难道在等被圈子先抛弃？

**和菜头回答要点：**
- 人际关系没那么简单：有意气相投的圈子，也有相互利用、互利互惠的圈子
- 存在身份差异和地位差异，不同人选择不同处理方式
- **不守时的人还在，说明圈子对他的需求很高**，所以能忍耐——变相承认他拥有某种特权
- 抛弃不可能，除非维系圈子的核心价值遭到破坏或地位发生转变
- 大家会忍耐下去直到彻底接受，变成习惯
- **人也是可以驯化的**——对此不要感到太过惊奇

**用户盒子（朋友不守时让自己难过）：**
朋友不守时→反思→发现关系不对等→见面心里膈应，怎么办？

**和菜头回答要点——"坦率谈话"测试法：**
- 坦率和对方谈一次：不批评、不指责，**只说清楚自己的感受**
  - "我每次都要等你，这让我非常不愉快"
  - "我觉得我没有得到应有的尊重"
  - "希望大家以后约见面能够准点到"
- **如果对方把你当朋友且尊重你**→会道歉，今后注意到
- **如果对方认为"这么小的事也值得说"**→说明你不是朋友——"这么小的人物也敢要求什么尊重"
- 风险：可能一拍两散——问题回到你：**你怕不怕一拍两散？**
- 这时换你去衡量对方的地位和价值

**用户肖晓北（早到的人 ⭐ 本期主题）：**
生活中总有人过分早到，经常给人带来困扰。这也算不守时吧？怎么看待，怎么对待？

**和菜头回答要点：**
- **早到的人不会是坏人**，我相信是这样的
- 自己和朋友们总是**提前五分钟到**，如果更早就周围散步转几圈，以免太早到给人压力或困扰
- **为什么早到的人不会是坏人？**
  - 传统上中国人认为太早到（尤其吃饭时）是**失仪**——表示自己太过热切想吃这顿饭
  - 说明**馋或者穷**
  - 如果连这个理都不知道，或者知道了并不在乎——这样的人**能坏到哪里去？**
- 朋友圈子里总得有几个**不谙世事、天真烂漫**的人
- 至于那些小毛病，**不妨大而化之**

**核心态度**：早到虽然也给人困扰，但出发点是**淳朴的天真**而非恶意。对早到的人，和菜头选择宽容——这种"不知道规矩/不在乎规矩"恰恰说明他们没有心机。

**用户朱军（不守时代表态度）：**
曾有一段时间非常不守时，那时非但不认同自己也不认同周遭，好在抑郁边缘徘徊不算太久。现在很守时，周遭也大多是守时的人。所以不守时可能代表一种态度，是吗？

**和菜头回答要点：**
- 守时对一类人非常重要，同类聚集→步调一致→减少抱怨摩擦→形成共同价值
- 不守时的人**也能形成自己的小群落**——因为这个群落存在其他共同价值，守时刚好不在其中
- **反对从道德层面讨论守时**：会增加对立而不是相互理解
- 你的情况更特殊：精神处于非常困苦状态时，自然无心关注守时细节——也说明**守时并非人性中非常基本的属性**，没有看重它的人想象的那么重要

**用户核桃（守时=同频）：**
守时就是和对方保持同频。遇到别人不守时就不舒服，以前劝自己"太多人没时间观念是我心理问题"。今天才认识：不愿守时=不愿同频。是不是矫枉过正？

**和菜头回答要点：**
- "愿意不愿意"是主观判断，我不知道你周围的人
- 保守地说：他们在**行为上无法和你同步，同时也不在意这种异步**——这是事实
- 从对方角度：守时会带来心理压力——有个守时的朋友会给自己带来心理压力
- 但这种压力不足以让他们改变行为→说明**你给他们带来的压力大于他们给你带去的压力**
- **反过来说**：这折射了你在人际圈子里**重要性的高低**——这才是你更应该思考的

## 核心洞察

| 主题 | 核心洞察 |
|------|---------|
| **最小成就感** | 不要从90到95，要从45到60。不反复投入已擅长的领域 |
| **选择的妄念** | "如果当时跳了会更好"是妄念。天真地相信自己的选择就是最好的 |
| **精神→身体** | 精神痛苦必然带来肉体痛苦，是常识。适应力会帮你逐渐减弱 |
| **不守时=高需求** | 不守时的人还在圈子里，说明圈子对他的需求很高，变相承认特权 |
| **坦率谈话测试** | 不批评不说教只谈感受——快速测试对方是否真把你当朋友 |
| **早到≠坏人** | 早到是天真烂漫、不谙世事，小毛病不妨大而化之 |
| **守时非基本属性** | 精神困苦时自然无心守时，守时没有看重它的人想象的那么重要 |
| **压力折射重要性** | 你给对方的守时压力>对方给你的→说明你在圈子里不如他重要 |
