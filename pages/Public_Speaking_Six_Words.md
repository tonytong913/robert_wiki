# Public_Speaking_Six_Words（演讲恐惧六字诀：不要脸，不是人）

克服公开演讲恐惧的秘诀只有六个字——**不要脸，不是人**。来自和菜头数百场演讲的亲身经验。

---

## 一句话定义

**"不要脸，不是人"** = 上台前破除对自身形象的执念（不要脸），台上不把观众当人看而是当桌椅板凳（不是人），以此消除演讲恐惧。

---

## 六字诀详解

### 不要脸（上台前）

| 恐惧来源 | 破除方法 |
|---------|---------|
| 担心忘词 | 真忘了才知道没那么严重 |
| 担心嘘声 | 观众多半是鼓励和期待态度 |
| 担心显得傻 | 想想为什么是你而不是别人上台 |
| 担心丢丑 | 一路不要脸想下去——最后总是鼓掌结束 |

> 上场之前自己会担心忘词、担心观众嘘声、担心自己看上去傻里傻气的——这就是**要脸的副作用**。

> 为什么是你而不是别人上台演讲？那肯定还是因为你有这个价值、有这个地位、有这个号召力。

### 不是人（台上）

| 做法 | 效果 |
|------|------|
| 把观众当桌椅板凳 | 心理压力骤减，就像在家对家具备稿 |
| 不要逐一分析观众表情 | 否则压力会天大 |
| 发现控场能力后 | 试着转头看观众——他们会随你的动作转 |
| 喜欢上控场感后 | 尝试互动，气氛更好更亲切 |

> 最起码在早期应该把观众当做是桌椅板凳，不是真实的存在。

---

## 进阶：从恐惧到控场

```
不要脸地走上去 → 备稿（观众=桌椅板凳）→ 
试着看观众 → 发现他们有反应、跟着你转 →
发现自己有控场能力 → 压力减少 → 
喜欢上这种感觉 → 主动互动 →
现场气氛更好 
```

> 类似这样做成一次，那之后的演讲就会好办得多了。

---

## 和菜头的亲身经历

做过数百场公众演讲，经历过：
- 头脑空白
- 痉挛
- 呕吐

> 事情根本就没有你想的那么严重。

---

## 补充：专业建议

演讲术在西方是很早就有的专门学科。哈佛商学院等有专门的公开课。六字诀是个人经验，建议也向专业人士学习。

---

## 关键洞察

1. **不要脸** — 上台前破除形象执念：为什么是你上台？因为你有价值
2. **不是人** — 台上把观众当家具：先自在地讲，再逐步互动
3. **做成一回就行** — 一次成功的控场经验，之后就此打开
4. **丢丑没那么严重** — 真丢了才知道事情没那么可怕
5. **演讲是学得会的** — 有专门学科，不要只靠自己硬扛
