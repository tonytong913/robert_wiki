# 真高管

> 真高管 ≠ 头衔上的CXO。真高管是那些在所辖领域，对内可以领导一号位及高管团队，对外可以代表公司竞争力的人。其精神底色是创业者。

---

## 关键要点

- **头衔不等于本质**：CXO、VP只是头衔，拥有这些头衔不等于就是真高管
- **反向也成立**：没有头衔的人（总监、独立贡献者），满足条件也是真高管
- **两条衡量标准**：
  1. **对内**——能领导一号位及高管团队
  2. **对外**——能代表公司的竞争力
- **精神底色是创业者**：真高管不是"高级打工人"心态，而是"创业者"心态
- **中小公司的现实**：大量公司没有CXO/VP，总监承担高管职责，照样发展很好

---

## 详细内容

### 定义来源：首席组织官

首席组织官（Chief Organization Officer）对真高管的正式定义：

> 那些在所辖领域，**对内可以领导一号位及高管团队，对外可以代表公司竞争力的人**。

这个定义有三个关键要素：

| 要素 | 含义 |
|------|------|
| **所辖领域** | 有自己负责的专业/业务领域，不是"万金油" |
| **对内领导一号位及高管团队** | 在自己的领域内，能引导、影响最高决策者和同级高管 |
| **对外代表公司竞争力** | 在外部（客户、市场、行业）能展现公司的竞争实力 |

### 为什么"胜任的CXO"不等于真高管

1. **大量公司根本没有CXO/VP**：中小公司、成长型公司，组织结构扁平，没有那么多头衔
2. **总监在干高管的活**：这些公司的总监承担了战略决策、跨部门协调、对外代表等高管职责
3. **结果证明一切**：这些公司有些发展得很好，说明"非CXO"也能有效履行高管职能
4. **头衔是外部标签，能力是内在实质**：不能因为一个人没有CXO头衔就说他不是高管

### 独立贡献者也可以是高管

最具颠覆性的观点：**即使没有下属、没人向你汇报，只要满足条件，你就是真高管。**

这意味着：
- 管理幅度不是衡量高管的标准
- 真正的衡量标准是**影响力和代表力**
- 一个技术大牛、一个顶级销售，他们虽然不管理人，但在专业领域引领方向、对外展现竞争力，他们就是高管

### 创业者精神：真高管的精神底色

真高管和"高级经理人"的关键差异在于：

| 高级经理人 | 真高管 |
|-----------|--------|
| 执行者心态 | 创业者心态 |
| 等待指令 | 主动承担责任（"我来"） |
| 管好自己一亩三分地 | 对内领导、对外代表 |
| 对职位负责 | 对公司竞争力负责 |

这与刘澜领导力口诀"[[Leadership/L03_Mantra_1_I_Come|我来]]"高度呼应——真高管不是等谁来任命，而是主动站出来承担。
