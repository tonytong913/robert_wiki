# Workplace_Manipulation_Tactics（职场操纵伎俩：节能型高管）

**有一种"节能型"高管——以拥护为名向上责任转移，以培养为由向下滥授权。他们只需动动嘴、刷刷脸，就能巧妙搭便车，轻松获取本不属于自己的成果和利益。**

---

## 一句话定义

节能型高管 = 不创造价值但善于截取价值的职场玩家——靠操纵上下级关系而非实际能力生存和晋升。

---

## 二合一技俩的完整图谱

```
             向上：以拥护为名
       故意留漏洞 → CEO"指导"修订
              ↓
       出了问题 → "是按CEO意见做的"
              ↓
           责任成功转移 ↑


             向下：以培养为由
       不考虑能力 → 抛复杂任务给员工
              ↓
       员工失败 → "给你机会你不中用"
              ↓
           责任成功转移 ↓


      高管 → 0实际贡献 + 100%利益获取
```

---

## 节能型高管的行为特征

| 特征 | 表现 |
|------|------|
| **动嘴不动手** | 从不亲自做任何实质工作 |
| **刷脸不担责** | 在领导面前刷存在感，但从不承担后果 |
| **搭便车** | 项目的成果归他，项目的失败是你的 |
| **话术精湛** | "拥护领导""培养员工"等正面词语掩盖 |

> "这些高管在企业当中只要稍微动动嘴刷刷脸，就能巧妙的搭便车，轻松的获取本不属于他的成果和利益。"

---

## 为什么他们能在组织中生存？

| 原因 | 分析 |
|------|------|
| **向上讨好有效** | CEO喜欢被"拥护"的感觉 |
| **向下伪装有效** | "培养员工"是政治正确的话术 |
| **责任链断裂** | 出问题找不到明确的负责人 |
| **绩效难以量化** | 管理岗位本就不易衡量实际贡献 |
| **组织惯性** | 换掉一个"看起来不错"的高管成本高 |

---

## 与真正领导力的对照

| 维度 | 节能型高管 | 真正的领导者 |
|------|-----------|------------|
| **向上** | 诱导、推责 | 负责、承担 |
| **向下** | 滥授权、抛弃 | 培养、陪伴 |
| **成果** | 窃取他人 | 创造价值 |
| **失败** | 永远不是我的错 | 是我没做好 |
| **话术** | 以拥护/培养为名 | 以"我来"/"我们"为实 |
| **对标页面** | — | [[Executive_Feet_in_the_Mud\|双脚沾泥]] |

## 如何应对

### 作为CEO
1. **区分拥护和推责** — 真心拥护不会把失败推给你
2. **看实际产出** — 剥离下属的贡献，看高管自己做了什么
3. **追问"如果是你，你会怎么做"** — 不给甩锅空间
4. **要求完整的责任归属** — 方案谁负责就是谁负责，不被"指导意见"绑架

### 作为员工
1. **识别滥授权信号** — 任务远超能力、无支持、无陪伴
2. **保护自己的成长路径** — 不被"机会"的糖衣迷惑
3. **记录和沟通** — 明确表达能力边界，留下书面记录

### 作为组织
1. **建立责任追溯机制** — 每个项目的责任人不得转移
2. **量化管理贡献** — 不能只用"领导认可"做评价
3. **保护说实话的人** — 节能型高管最怕被看穿

---

## 关键启示

1. **节能型高管是组织的寄生者** — 自己不创造价值，只截取他人的成果
2. **"拥护"和"培养"是最高明的伪装** — 正面词语掩盖了负面行为
3. **组织的沉默是他们的温床** — 不被戳破就能一直存在
4. **真正的高管双脚沾泥** — 责任不转移、授权不滥发、价值不窃取
5. **动嘴刷脸的成本极低，收益极高** — 这是他们不断克隆自己的动力
