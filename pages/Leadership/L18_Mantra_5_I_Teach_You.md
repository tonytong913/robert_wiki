## 一、核心概念

### 1.1 "我教你"——领导者的育人角色

在刘澜领导力课程中，**"我教你"是第五句领导力口诀**。这句口诀强调的是领导者作为老师的育人角色，即领导者需要通过教导来培养下属。

#### 为什么领导者要当老师

| 维度 | 说明 |
|------|------|
| **组织发展** | 组织的持续发展需要人才梯队 |
| **下属成长** | 下属需要指导才能更快成长 |
| **知识传承** | 经验和技能需要传递下去 |
| **解放自己** | 培养人才能让自己从具体事务中解脱 |

### 1.2 "我教你"的五种说法

围绕"我教你"这句领导力口诀，领导者当老师有**五个层次**，分别对应"我教你"的**五种具体说法**：

```
┌─────────────────────────────────────────────────────────┐
│              "我教你"的五种说法                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   1. "照我说的做"  →  管教（管理层面）                  │
│                                                         │
│   2. "我说给你听"  →  说教（管理层面）                  │
│                                                         │
│   3. "我做给你看"  →  身教（管理层面）                  │
│                                                         │
│   4. "你说怎么做"  →  请教（领导层面）★                │
│                                                         │
│   5. "你为什么做"  →  传教（领导层面）★                │
│                                                         │
│   ★ 表示更高层次的领导行为                              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 二、五个层次的详细解释

### 2.1 第一层次：管教——"照我说的做"

#### 核心含义

**管教**对应"照我说的做"，这是最基础的教导方式，更多是在**管理层面**。

#### 特点与应用

| 特点 | 说明 | 适用场景 |
|------|------|---------|
| **指令明确** | 直接告诉下属做什么 | 紧急情况、新手入门 |
| **强调服从** | 要求按标准执行 | 流程规范、安全要求 |
| **快速高效** | 不需要过多解释 | 时间紧迫、任务简单 |

#### 管教方式的利弊

```
┌─────────────────────────────────────────────────────────┐
│                  管教的利弊分析                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   优点：                                                │
│   ✓ 执行效率高                                          │
│   ✓ 标准统一                                            │
│   ✓ 适合简单重复性工作                                  │
│                                                         │
│   局限：                                                │
│   ✗ 下属被动执行，缺乏主动性                            │
│   ✗ 不利于培养思考能力                                  │
│   ✗ 长期依赖会抑制创新                                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 使用建议

- 适合**新员工**或**简单任务**
- 需要配合**解释原因**，避免机械执行
- 随着下属成长，逐步减少使用

---

### 2.2 第二层次：说教——"我说给你听"

#### 核心含义

**说教**对应"我说给你听"，通过语言向下属传授知识和道理，更多是在**管理层面**。

#### 特点与应用

| 特点 | 说明 | 适用场景 |
|------|------|---------|
| **知识传递** | 讲解概念、原理、方法 | 培训、会议 |
| **经验分享** | 讲述自己的经历和教训 | 案例教学 |
| **道理阐述** | 解释为什么这样做 | 价值观传递 |

#### 说教方式的利弊

```
┌─────────────────────────────────────────────────────────┐
│                  说教的利弊分析                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   优点：                                                │
│   ✓ 信息传递量大                                        │
│   ✓ 可以系统讲解知识                                    │
│   ✓ 适合理论性内容                                      │
│                                                         │
│   局限：                                                │
│   ✗ 容易变成单向灌输                                    │
│   ✗ 下属可能左耳进右耳出                                │
│   ✗ 缺乏实践环节                                        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 提升说教效果的方法

- **结合案例**：用具体例子说明抽象道理
- **互动提问**：让下属参与讨论
- **及时反馈**：检查理解程度
- **跟进实践**：将所学应用到工作中

---

### 2.3 第三层次：身教——"我做给你看"

#### 核心含义

**身教**对应"我做给你看"，通过示范让下属学习，更多是在**管理层面**。

#### 特点与应用

| 特点 | 说明 | 适用场景 |
|------|------|---------|
| **示范引领** | 领导者亲自做示范 | 技能传授、标准演示 |
| **行为标杆** | 以身作则树立榜样 | 文化建设、价值观践行 |
| **直观易学** | 下属可以直接观察模仿 | 操作性强的任务 |

#### 身教的威力

```
┌─────────────────────────────────────────────────────────┐
│                  身教胜于言传                            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   "我做给你看"的力量：                                  │
│                                                         │
│   • 比"我说给你听"更有说服力                           │
│   • 下属可以看到具体做法                                │
│   • 建立了可模仿的标杆                                  │
│   • 体现了领导者的承诺和投入                            │
│                                                         │
│   关键原则：                                            │
│   要求下属做到的，自己首先要做到                        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 身教的实践要点

1. **明确示范目标** —— 让下属知道要看什么
2. **讲解关键步骤** —— 边做边说明要点
3. **允许提问** —— 鼓励下属提出疑问
4. **让下属尝试** —— 看过之后亲自实践
5. **给予反馈** —— 及时指出改进点

---

### 2.4 第四层次：请教——"你说怎么做"

#### 核心含义

**请教**对应"你说怎么做"，这是更高层次的教导方式，更多是在**领导层面**，真正把对方改到主角位置。

#### 与前三个层次的本质区别

```
┌─────────────────────────────────────────────────────────┐
│              从"我主导"到"你主导"的转变                 │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   前三个层次（管教、说教、身教）：                      │
│   ┌─────────────────────────────────────────┐          │
│   │                                         │          │
│   │   领导者是主角 ←──── 下属是配角         │          │
│   │   "我来教你"      "你跟着我学"          │          │
│   │                                         │          │
│   └─────────────────────────────────────────┘          │
│                                                         │
│   第四层次（请教）：                                    │
│   ┌─────────────────────────────────────────┐          │
│   │                                         │          │
│   │   下属是主角  ←──── 领导者是配角        │          │
│   │   "你说怎么做"    "我听听你的想法"      │          │
│   │                                         │          │
│   └─────────────────────────────────────────┘          │
│                                                         │
│   关键转变：从"我告诉你"到"我听你说"                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 请教的核心价值

| 价值 | 说明 |
|------|------|
| **培养主动性** | 下属需要主动思考解决方案 |
| **激发创造力** | 不限制下属的思维方式 |
| **增强责任感** | 方案是下属提出的，执行更有动力 |
| **发现人才** | 通过下属的想法发现其潜力 |

#### 请教的实践方法

```
┌─────────────────────────────────────────────────────────┐
│                  "你说怎么做"的实践                      │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   1. 布置任务时：                                        │
│      "这个任务交给你，你先想想怎么做，然后告诉我"       │
│                                                         │
│   2. 下属遇到困难时：                                    │
│      "你遇到了什么困难？你觉得可以怎么解决？"           │
│                                                         │
│   3. 做决策时：                                          │
│      "对于这个问题，你有什么建议？"                     │
│                                                         │
│   4. 复盘总结时：                                        │
│      "这次项目，你觉得哪些地方可以改进？"               │
│                                                         │
│   关键心态：                                            │
│   • 真诚地想听下属的想法                                │
│   • 不轻易否定，先理解再引导                            │
│   • 即使想法不成熟，也要给予肯定和鼓励                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

---

### 2.5 第五层次：传教——"你为什么做"

#### 核心含义

**传教**对应"你为什么做"，这是最高层次的教导方式，更多是在**领导层面**，关注事情的**意义和目标**。

#### 传教的本质

```
┌─────────────────────────────────────────────────────────┐
│                  传教的本质                              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   "你为什么做"关注的是：                                │
│                                                         │
│   • 做这件事的**目的**是什么？                          │
│   • 这件事的**意义**何在？                              │
│   • 这件事与**更大目标**的关系？                        │
│   • 这件事对**个人成长**的价值？                        │
│                                                         │
│   核心：不仅教"怎么做"，更教"为什么做"                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 传教与前四个层次的区别

| 层次 | 关注点 | 问题 |
|------|--------|------|
| 管教 | 行为 | "做什么" |
| 说教 | 知识 | "是什么" |
| 身教 | 技能 | "怎么做" |
| 请教 | 思考 | "你想怎么做" |
| **传教** | **意义** | **"为什么做"** |

#### 传教的核心价值

```
┌─────────────────────────────────────────────────────────┐
│              "你为什么做"的价值                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   1. 激发内在动力                                       │
│      当人理解了"为什么"，会更主动投入                   │
│                                                         │
│   2. 建立意义感                                         │
│      让下属感到工作有价值、有意义                       │
│                                                         │
│   3. 增强使命感                                         │
│      将个人工作与更大目标连接                           │
│                                                         │
│   4. 培养战略思维                                       │
│      让下属学会从更高视角看问题                         │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 传教的实践方法

| 场景 | "你为什么做"的应用 |
|------|-------------------|
| **任务布置** | 解释任务对团队/公司的意义 |
| **目标设定** | 说明目标背后的战略意图 |
| **价值观传递** | 解释"我们为什么坚持这些原则" |
| **变革推动** | 说明变革的必要性和长远价值 |
| **困难鼓励** | 帮助下属看到困难中的成长意义 |

## 三、五个层次的关系

### 3.1 层次高低之分

这五个层次有**高低之分**：

```
┌─────────────────────────────────────────────────────────┐
│              五个层次的进阶关系                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   低层次 ──────────────────────────────→ 高层次        │
│                                                         │
│   管教 → 说教 → 身教 → 请教 → 传教                     │
│    ↓      ↓      ↓      ↓      ↓                       │
│   "照    "我说   "我做   "你说   "你为什么             │
│    我说的  给你听  给你看  怎么做"  做"                 │
│    做"                                                  │
│                                                         │
│   管理层面 ───────────────→ 领导层面                   │
│                                                         │
│   注：箭头表示层次提升，但不表示取代关系               │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 3.2 不能相互取代

虽然五个层次有高低之分，但**不能相互取代**，就像**五种工具**，不同场合需要用不同工具。

```
┌─────────────────────────────────────────────────────────┐
│              五种工具，各有所用                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   ┌─────────────────────────────────────────────────┐  │
│   │                                                 │  │
│   │   就像工具箱里有锤子、螺丝刀、扳手...           │  │
│   │   每种工具都有其适用场景                        │  │
│   │                                                 │  │
│   │   领导者需努力掌握这五种工具                    │  │
│   │   根据情境灵活选择                              │  │
│   │                                                 │  │
│   └─────────────────────────────────────────────────┘  │
│                                                         │
│   关键认知：                                            │
│   • 高层次不等于"总是更好"                            │
│   • 低层次在特定场景下更有效                          │
│   • 优秀的领导者会根据情况灵活切换                    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 3.3 五个层次的分组

```
┌─────────────────────────────────────────────────────────┐
│              五个层次的两组划分                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   第一组：管理层面                                       │
│   ┌─────────────────────────────────────────┐          │
│   │  管教  │  说教  │  身教                 │          │

│   │ "照    │ "我说  │ "我做                 │          │

│                                                         │
│   第二组：领导层面                                       │

│   │  请教  │  传教                         │          │

```

## 四、如何选择合适的层次

### 4.1 选择依据

| 考虑因素 | 适用层次 |
|---------|---------|
| **下属能力** | 能力弱→管教/说教；能力强→请教/传教 |
| **任务性质** | 简单重复→管教；复杂创新→请教 |
| **时间压力** | 紧急→管教/身教；从容→请教/传教 |
| **发展阶段** | 新员工→管教/说教；老员工→请教/传教 |
| **目标类型** | 执行目标→身教；发展目标→传教 |

### 4.2 情境选择示例

```
┌─────────────────────────────────────────────────────────┐
│              不同情境下的层次选择                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   情境1：新员工第一次做报表                              │
│   → 选择：身教（"我做给你看"）                         │
│   → 理由：需要直观学习操作步骤                          │
│                                                         │
│   情境2：项目遇到突发问题，需要立即解决                  │
│   → 选择：管教（"照我说的做"）                         │
│   → 理由：时间紧迫，需要快速执行                        │
│                                                         │
│   情境3：培养储备干部的战略思维                          │
│   → 选择：传教（"你为什么做"）                         │
│   → 理由：需要理解决策背后的意义                        │
│                                                         │
│   情境4：激发资深员工的创新潜力                          │
│   → 选择：请教（"你说怎么做"）                         │
│   → 理由：发挥其经验和创造力                            │
│                                                         │
│   情境5：传达公司新的价值观                              │
│   → 选择：说教+传教（"我说给你听"+"你为什么做"）      │
│   → 理由：既需要解释内容，也需要说明意义                │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 五、领导力行动口诀

### 5.1 口诀总结

> **"我教你"对应的领导力修炼是当老师，领导者需要在五个层次上当老师。**

### 5.2 修炼要点

```
┌─────────────────────────────────────────────────────────┐
│              "我教你"的修炼要点                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   1. 掌握五种说法                                       │
│      • 能够灵活运用"照我说的做""我说给你听"           │
│        "我做给你看""你说怎么做""你为什么做"           │
│                                                         │
│   2. 理解层次差异                                       │
│      • 知道五个层次的区别和适用场景                     │
│      • 明白高层次不等于总是更好                         │
│                                                         │
│   3. 根据情境选择                                       │
│      • 根据下属、任务、时间等因素灵活选择               │
│      • 避免一刀切的做法                                 │
│                                                         │
│   4. 持续练习提升                                       │
│      • 有意识地使用不同层次的教导方式                   │
│      • 反思效果，不断改进                               │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 六、行动题

### 6.1 实践任务

> **使用"我教你"这句口诀的五种说法之一去实践当老师这项领导力修炼，然后分享实践结果。**

### 6.2 实践建议

| 如果你平时多用... | 建议尝试... | 实践场景 |
|------------------|------------|---------|
| 管教（"照我说的做"） | 请教（"你说怎么做"） | 让下属主导一个小项目 |
| 说教（"我说给你听"） | 身教（"我做给你看"） | 亲自示范一项技能 |
| 身教（"我做给你看"） | 传教（"你为什么做"） | 解释工作的意义和价值 |
| 请教（"你说怎么做"） | 说教（"我说给你听"） | 系统分享你的经验 |
| 传教（"你为什么做"） | 管教（"照我说的做"） | 紧急情况下果断指令 |

### 6.3 反思问题

实践后思考以下问题：

1. 你选择了哪种说法？为什么选择它？
2. 实践过程中遇到了什么挑战？
3. 下属的反应如何？

## 八、关键要点总结

### 8.1 核心观点

1. **"我教你"是第五句领导力口诀** —— 强调领导者的育人角色
2. **五种说法对应五个层次** —— 管教、说教、身教、请教、传教
3. **五个层次有高低之分** —— 但不能相互取代
4. **两组划分** —— 前三者在管理层面，后两者在领导层面
5. **灵活运用** —— 根据情境选择合适的教导方式

### 8.2 五句口诀速查

| 口诀 | 核心含义 | 应用场景 |
|------|---------|---------|
| "我来" | 承担责任 | 需要挺身而出时 |
| "我不知道" | 保持谦逊 | 面对不确定时 |
| "你觉得呢" | 动员群众 | 需要集思广益时 |
| "我讲个故事" | 建立连接 | 传递价值观时 |
| **"我教你"** | **当老师** | **培养下属时** |

### 8.3 一句话总结

> **"我教你"不是简单的知识传授，而是在五个层次上因材施教：从"照我说的做"到"你为什么做"，领导者需要根据情境灵活运用这五种教导方式。**
