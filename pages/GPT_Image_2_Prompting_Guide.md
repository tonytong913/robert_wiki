# GPT Image 2 提示词编写指南

供人类和 AI 参考的 GPT Image 2 提示词最佳实践。GPT Image 2 是 OpenAI 2026 年 4 月发布的第二代图片生成模型，支持 Thinking Mode、95%+ 文字渲染精度、最多 16 张参考图、原生 4K 输出。

---

## 核心公式

```
[主体 + 形容词] [动作] [场景/背景]
[构图/镜头]
[光线/氛围]
[风格/媒介]
[精确文字 + 排版]
[用途/比例]
```

GPT Image 2 奖励**清晰的自然语言描述**，不需要 `8k masterpiece` 这类关键词堆砌。

---

## 十大核心原则

### 1. 结构化 + 明确目标
固定顺序（场景 → 主体 → 细节 → 约束），标注用途以设定模式和精美程度。

### 2. 具体材质与媒介
声明材质（哑光/光泽）、视觉媒介（照片/水彩/3D）、画质关键词（胶片颗粒/微距）。写实场景直接包含 `photorealistic`。

### 3. 不用关键词堆砌
>`8k, masterpiece, trending on artstation` 对 GPT Image 2 无效，它读自然语言。

### 4. 构图与光线控制
景别（特写/全景）、角度（平视/仰角）、镜头（50mm/85mm）、光线（金色时刻/霓虹/柔和散射）、布局（"logo 右上角"）。

### 5. 人物控制
描述比例、取景框、视线方向、手部动作：`全身可见、低头看书不看镜头、双手自然握把`。

### 6. 约束与保留
生成时：`无水印、无额外文字、无 logo`。编辑时：`只改 X，保留身份/几何/布局/饱和度`。

### 7. 图片内文字
精确文字用引号或全大写：`"SUMMER COLLECTION"`，指定字体风格/大小/颜色/位置。生僻词逐字母拼写。密集文字用 `quality=high`。

### 8. 多图输入
用索引描述每张图：`图1：产品照片…图2：风格参考…`，描述交互：`将图2风格应用到图1`。

### 9. 迭代而非一次塞满
先写干净基础提示词 → 小的单点修改迭代 → 每次编辑重复保留列表防止漂移。

### 10. 质量 vs 速度
- `quality=low`：高吞吐、草稿，多数场景已够用
- `quality=medium`：一般生产
- `quality=high`：密集文字、人像特写、身份敏感编辑

## 十大实用模板

1. **产品广告**：干净背景+工作室灯光+投影+标语
2. **电商带文字**：写实材质+精确标签文字+字间距
3. **细节特写人像**：极限特写+浅景深+85mm+写实皮肤
4. **多参考图合成**：图1角色+图2服装+图3姿势+新场景
5. **品牌指南**：logo+色板+字体+社媒帖+包装模型
6. **信息图**：真实信息+矢量风格+柔和色板+9:16
7. **杂志拼贴**：4-6张照片+撕纸边+手写批注+标题
8. **App 截图**：真实 UI 元素+精确消息内容+无水印
9. **电影级宽幅**：21:9+体积光+大气雾霾+史诗尺度
10. **JSON 结构化**：锁定参数的可重复生成

## 提示词模板（供 AI 调用）

```
任务：[产品广告/信息图/概念艺术/人像/UI]
主体：[品类+核心特征+材质]
场景：[环境+背景]
构图：[景别+角度+镜头焦段]
光线：[类型+方向+色温]
风格：[摄影/3D/插画/矢量+具体风格]
文字：[精确内容+字体+位置]
约束：[排除项+保留项]
比例：[1:1/9:16/16:9/21:9]
```

## 常见错误

| 错误 | 正确 |
|------|------|
| 关键词堆砌 | 自然语言描述 |
| "加个标题" | 精确指定文字+字体+位置+颜色 |
| 默认 4K | quality=low 先测试 |
| 编辑忘保留列表 | 每次重复 `保留：身份/光照/布局` |
