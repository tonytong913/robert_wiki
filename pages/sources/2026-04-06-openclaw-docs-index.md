## 文档概览

OpenClaw 是一个自托管的多通道 AI Agent 网关，支持 Discord、Telegram、WhatsApp、QQ 等 20+ 聊天平台。

**核心特点:**
- 自托管: 运行在自己的硬件上
- 多通道: 一个 Gateway 服务多个聊天平台
- Agent 原生: 支持工具调用、会话、记忆、多 Agent 路由
- 开源: MIT 许可证
