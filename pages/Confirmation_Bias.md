# 确认偏误

确认偏误（Confirmation Bias）是指人们倾向于寻找、解释和记忆那些能够证实自己已有信念或假设的信息，同时忽视或低估反面证据的心理倾向。

---

## 核心机制

### 1. 选择性注意

- 更容易注意到支持自己观点的信息
- 对反面信息视而不见或快速遗忘
- 在复杂信息中"发现"自己想要的模式

### 2. 选择性解释

- 对模糊信息做有利于自己观点的解读
- 对同一证据，支持方和反对方都能从中找到支持
- 用不同标准评估支持性和反对性证据

### 3. 选择性记忆

- 更容易记住证实自己观点的经历
- 遗忘或淡化与自己观点冲突的证据
- 回忆时无意中扭曲记忆以符合当前信念

---

## 常见表现

| 场景 | 表现 |
|------|------|
| 新闻阅读 | 只关注符合自己立场的媒体 |
| 社交互动 | 与观点相似的人交往更多 |
| 决策过程 | 寻找支持已偏爱选项的理由 |
| 争论中 | 只接受能证明自己正确的论据 |
| 学习过程 | 偏爱证实已有知识的内容 |

---

## 为什么难以避免

### 心理根源

1. **认知节省**：处理所有信息太耗费精力
2. **自我认同**：观点与自我紧密绑定
3. **社会归属**：观点与群体身份相关
4. **确定性需求**：不确定感带来焦虑

### 强化机制

- **算法推荐**：信息茧房效应
- **社交过滤**：朋友圈同质化
- **自我实现**：信念影响行为，行为强化信念

---

## 应对策略

### 个人层面

#### 1. 主动寻找反面证据

- 刻意阅读对立立场的文章
- 问自己："什么证据能让我改变观点？"
- 建立"反方文件夹"

#### 2. 改变提问方式

| 避免 | 改用 |
|------|------|
| "为什么我是对的？" | "为什么我可能错了？" |
| "支持我的证据是什么？" | "反驳我的证据是什么？" |
| "谁同意我？" | "谁不同意我，为什么？" |

#### 3. 延迟判断

- 重要决策前强制等待期
- 先收集信息，再形成观点
- 避免"第一印象决定一切"

#### 4. 外部视角

- 咨询持不同观点的人
- 建立"红队"机制（专门唱反调）
- 使用事前验尸法（Pre-mortem）

### 系统层面

- **多元化信息源**：主动接触不同立场
- **异质社交网络**：与不同背景的人交往
- **制度化质疑**：组织中的"魔鬼代言人"

---

## 与局限的关系

确认偏误是 [[Cognitive_Limitation|认知局限]] 自我强化的核心机制：

```
匮乏 → 形成局限 → 确认偏误 → 强化局限 → 发展为"真理"
```

突破局限的关键就是**打破确认偏误的循环**。

---

## 在决策中的影响

### 投资领域

- 只看到支持自己持仓的利好消息
- 忽视风险信号
- 导致过度自信和损失

### 医疗领域

- 医生可能过早确认初步诊断
- 忽视不符合诊断的症状
- 延误正确治疗

### 科学研究

- 研究者可能无意识地偏向预期结果
- 双盲实验的设计就是为了对抗确认偏误
