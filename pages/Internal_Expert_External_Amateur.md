# Internal_Expert_External_Amateur（内战内行，外战外行）

**有些高管在公司内部权斗中长袖善舞，一旦面对外部市场竞争却全然不能体现公司竞争力——这就是「内战内行，外战外行」。**

---

## 一句话定义

内战内行，外战外行 = 高管把精力消耗在内部权斗和资源争夺上，而失去了在所辖领域对外竞争的能力——内部是猛虎，外部是纸老虎。

---

## 核心现象

```
        对内（内战内行）              对外（外战外行）
    ┌─────────────────┐       ┌─────────────────┐
    │ 权力斗争 长袖善舞  │       │ 市场竞争 毫无建树  │
    │ 资源争夺 得心应手  │  ≠≠≠  │ 行业对标 无法代表  │
    │ 老资历 一览众山小  │       │ 客户面前 毫无力量  │
    └─────────────────┘       └─────────────────┘
```

---

## "内战内行"的特征

| 特征 | 表现 |
|------|------|
| **老资历压人** | 凭借年资在公司内部「一览众山小」 |
| **权斗高手** | 「长袖善舞」——在权力斗争中风生水起 |
| **资源掠夺者** | 「得心应手」——在资源争夺中占尽便宜 |
| **内部光环** | 在公司内部声望高、话语权重 |

这些能力全是**零和博弈**——一个人的赢建立在另一个人的输之上，不创造增量价值。

---

## "外战外行"的特征

| 特征 | 表现 |
|------|------|
| **竞争力缺失** | 在所辖领域不能代表公司竞争力 |
| **市场脱节** | 不了解行业动态和竞争对手 |
| **客户无感** | 无法在客户面前体现专业价值 |
| **创新无能** | 不会开拓新市场、新产品、新模式 |

---

## 双重伤害

### 对内部
> 「内部的优秀人才，就像被巨石压住的幼苗，难以成长。他们的光芒被压制，才华无处施展。」

| 伤害 | 机制 |
|------|------|
| **压制人才** | 内战内行的高管把精力放在打压下属上 |
| **幼苗窒息** | 有潜力的年轻人没有成长空间 |
| **逆向淘汰** | 能干的人离开，会来事的人留下 |

### 对外部
> 「外部的贤才看到这样的高管，也会望而却步，毕竟谁也不想陷入是非纷扰。」

| 伤害 | 机制 |
|------|------|
| **吓退贤才** | 外部优秀人才看到这样的高管就不想加入 |
| **人才黑洞** | 组织变成只进不出的单向通道 |
| **声誉损耗** | 行业口碑下降，招聘越来越难 |

### 对公司
> 「公司就像一辆陷入泥潭的车，发展必然受到极大的限制，难以驶向光明的未来。」

---

## 与 [[True_Executive|真高管]] 的直接对应

本文直接援引了真高管的定义：

> 「评判一个高管是否是'真高管'，不能只看职位、头衔、层级，而要以'能否在所辖领域对外对内代表公司竞争力'作为重要的衡量标准。」

| 维度 | 真高管 | 内战内行型高管 |
|------|--------|---------------|
| **对内** | 领导一号位和高管团队（影响力） | 权斗和资源争夺（权力） |
| **对外** | 代表公司竞争力 | 完全不能代表 |
| **能力方向** | 向外竞争，创造增量 | 向内争夺，分配存量 |
| **精神底色** | 创业者 | 官僚 |

---

## 根本原因

1. **路径依赖** — 靠内部权斗上位的人，能力模型固化在内部博弈
2. **舒适区** — 内部熟悉的规则让人安心，外部陌生的竞争让人恐惧
3. **无外部压力** — 公司考核偏重内部指标而非外部竞争力
4. **权力成瘾** — 内斗的「爽感」远大于外战的「苦感」

---

## 如何识别

| 信号 | 具体表现 |
|------|---------|
| **开会时间分配** | 80%时间讨论内部问题，20%讨论市场和客户 |
| **成功叙事** | 「我在公司推动了一个大项目」而非「我们在市场上打败了谁」 |
| **社交圈** | 几乎全是公司内部人，没有行业人脉 |
| **对标的对象** | 对标内部其他部门/人，而非外部竞争对手 |
| **恐惧来源** | 怕被内部人取代，不怕被市场淘汰 |

---

## 关键启示

1. **内战的胜利是虚幻的** — 内部赢了多少场，对外可能一场都赢不了
2. **真高管的核心标准是对外竞争力** — 没有外部竞争力，内部再能干也不是真高管
3. **压制人才=自毁长城** — 内斗型高管赶走的正是公司最需要的人
4. **组织需要向外看的考核标准** — 如果只考核内部指标，就会培养内战内行的人
5. **创业精神是解药** — 见 [[True_Executive|真高管]]：精神底色是创业者，不是高级打工人
