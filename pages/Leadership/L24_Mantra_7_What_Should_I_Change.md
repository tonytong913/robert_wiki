## 一、第七句口诀：我要改变什么？

### 1.1 口诀解读

> **"我要改变什么？"**

这是领导力修炼的**第七句口诀**，也是反思系列的核心口诀。它强调领导者需要主动思考并采取行动进行改变，而非被动等待或维持现状。

```
┌─────────────────────────────────────────────────────────┐
│              第七句口诀：我要改变什么？                 │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   核心含义：                                            │
│   • 主动思考 —— 不是被动接受，而是主动审视            │
│   • 发现问题 —— 识别需要改变的地方                    │
│   • 采取行动 —— 思考之后要有改变的行动                │
│                                                         │
│   关键问题：                                            │
│   • 我在做什么？                                        │
│   • 这样做对吗？                                        │
│   • 我需要改变什么？                                    │
│   • 如何改变？                                          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 1.2 为什么需要这句口诀？

领导力的本质之一是**推动改变**。没有改变的反思只是空想，没有反思的改变则是盲目。

```
┌─────────────────────────────────────────────────────────┐
│              反思与改变的关系                           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   ❌ 只有反思，没有改变                                 │
│   ─────────────────                                     │
│   • 陷入过度思考，成为"分析瘫痪"                      │
│   • 知道问题所在，但无所作为                          │
│   • 纸上谈兵，缺乏行动力                              │
│                                                         │
│   ❌ 只有改变，没有反思                                 │
│   ─────────────────                                     │
│   • 盲目行动，方向错误                                │
│   • 重复同样的错误                                    │
│   • 忙碌但无效                                        │
│                                                         │
│   ✅ 反思 + 改变 = 成长                                 │
│   ─────────────────                                     │
│   • 先思考，再行动                                    │
│   • 边行动，边反思                                    │
│   • 持续迭代，不断进步                                │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 二、反思的三个层次

### 2.1 小反思、中反思、大反思

反思可以分为三个层次：**小反思**、**中反思**和**大反思**。普通人常常只进行小反思，而优秀的领导者在关键时刻需要进行大反思。

```
┌─────────────────────────────────────────────────────────┐
│              反思的三个层次                             │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   层次1️⃣  小反思                                        │
│   ─────────────                                         │
│   范围：具体行为、方法、技巧                            │
│   问题："我这样做对吗？" "有更好的方法吗？"           │
│   示例：会议开得不好，下次改进议程安排                │
│   特点：表层改进，不改变根本                            │
│                                                         │
│   层次2️⃣  中反思                                        │
│   ─────────────                                         │
│   范围：策略、流程、制度                                │
│   问题："我们的策略对吗？" "流程需要调整吗？"         │
│   示例：销售策略效果不佳，调整目标客户群体            │
│   特点：结构性调整，改变做事方式                        │
│                                                         │
│   层次3️⃣  大反思                                        │
│   ─────────────                                         │
│   范围：心智模式、目标、方向、价值观                    │
│  问题："我们在做正确的事吗？" "我们的假设成立吗？"   │
│   示例：英特尔放弃内存业务，转向处理器                │
│   特点：根本性改变，改变思考方式                        │
│                                                         │
│   📊 反思层次对比                                       │
│   ┌──────────┬──────────┬──────────┐                  │
│   │  小反思  │  中反思  │  大反思  │                  │
│   ├──────────┼──────────┼──────────┤                  │
│   │  怎么做  │  做什么  │  为何做  │                  │
│   │  战术层  │  策略层  │  战略层  │                  │
│   │  改进    │  调整    │  变革    │                  │
│   └──────────┴──────────┴──────────┘                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 2.2 普通人的反思习惯

大多数人停留在**小反思**层面：

> **常见的小反思场景：**
> - "这次演讲准备不够充分，下次要提前准备"
> - "这个方案客户不满意，下次要多了解需求"
> - "会议效率太低，下次要控制时间"

这些小反思有价值，但**不足以带来根本性突破**。它们是在既定框架内的优化，而非框架本身的改变。

## 三、大反思的重要性

### 3.1 什么是大反思？

**大反思**涉及对**心智模式**的改变，包括目标、方向等深层次的思考。通过大反思，领导者能够发现并纠正自身及组织中的根本性问题。

```
┌─────────────────────────────────────────────────────────┐
│              大反思的核心特征                           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   特征1️⃣  质疑基本假设                                  │
│   ─────────────────                                     │
│   • 我们对市场的假设正确吗？                            │
│   • 我们的商业模式还成立吗？                            │
│   • 客户的需求变化了吗？                                │
│                                                         │
│   特征2️⃣  重新审视目标                                  │
│   ─────────────────                                     │
│   • 我们在追求的目标还值得追求吗？                    │
│   • 有没有更重要的目标被我们忽视了？                    │
│   • 目标本身是否需要改变？                              │
│                                                         │
│   特征3️⃣  改变心智模式                                  │
│   ─────────────────                                     │
│   • 我们如何看待这个世界？                              │
│   • 我们的认知框架是否需要更新？                        │
│   • 有没有全新的视角我们从未考虑过？                    │
│                                                         │
│   特征4️⃣  根本性变革                                    │
│   ─────────────────                                     │
│   • 不是改进，而是重新定义                              │
│   • 不是调整，而是彻底转型                              │
│   • 不是优化，而是创新突破                              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 3.2 大反思的时机

什么时候需要进行大反思？

| 信号 | 说明 | 示例 |
|-----|------|------|
| **业绩持续下滑** | 常规手段无法扭转 | 连续多个季度业绩下滑 |
| **环境重大变化** | 外部条件发生质变 | 新技术颠覆、政策变化 |
| **陷入瓶颈期** | 小改进无法突破 | 多年停留在同一水平 |
| **危机时刻** | 生死存亡关头 | 资金链断裂、核心客户流失 |
| **成功之后** | 巅峰时刻最容易忽视 | 市场领导者被颠覆 |

## 四、英特尔案例：大反思的力量

### 4.1 背景

20世纪80年代中期，英特尔公司面临严重的经营危机：

```
┌─────────────────────────────────────────────────────────┐
│              英特尔面临的困境                           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   📍 时间：1985年左右                                   │
│                                                         │
│   📍 核心业务：内存芯片（DRAM）                         │
│   ─────────────────────────────                         │
│   • 英特尔以内存芯片起家，这是公司的"根"              │
│   • 内存业务是公司身份的象征                            │
│   • 大量资源投入，技术积累深厚                          │
│                                                         │
│   📍 面临的挑战                                         │
│   ─────────────────                                     │
│   • 日本竞争对手（日立、NEC、富士通等）强势崛起       │
│   • 日本厂商质量更好、价格更低                          │
│   • 英特尔内存业务持续亏损                              │
│   • 公司陷入严重财务危机                                │
│                                                         │
│   📍 艰难抉择                                           │
│   ─────────────────                                     │
│   • 继续投入内存业务？——可能拖垮整个公司              │
│   • 放弃内存业务？——等于放弃公司根基                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 4.2 大反思的过程

在这个关键时刻，英特尔CEO安迪·格鲁夫（Andy Grove）进行了一次**大反思**。

> **格鲁夫的反思对话：**
> 
> 格鲁夫问董事长戈登·摩尔（Gordon Moore）：
> "如果我们被董事会踢出公司，新的CEO会怎么做？"
> 
> 摩尔毫不犹豫地回答：
> "新的CEO会退出内存业务。"
> 
> 格鲁夫追问：
> "那为什么我们不能自己这样做，然后走出办公室再回来呢？"

```
┌─────────────────────────────────────────────────────────┐
│              格鲁夫大反思的关键洞察                     │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   🔑 核心洞察                                           │
│   ─────────────                                         │
│   • 问题不是"内存业务还能挽救吗？"                    │
│   • 问题是"如果我们从零开始，还会做内存吗？"          │
│   • 新的CEO没有历史包袱，会做出更理性的决策           │
│                                                         │
│   🎯 反思的层次                                         │
│   ─────────────                                         │
│   这不是小反思（如何改进内存业务）                      │
│   这不是中反思（如何调整业务组合）                      │
│   这是大反思（我们是否还应该做内存）                    │
│                                                         │
│   💡 心智模式的转变                                     │
│   ─────────────────                                     │
│   从"我们是内存公司" → "我们是技术公司"              │
│   从"保护核心业务" → "追求最佳机会"                  │
│   从"过去的成功" → "未来的可能"                      │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 4.3 决策与结果

基于这次大反思，英特尔做出了艰难但正确的决定：

```
┌─────────────────────────────────────────────────────────┐
│              英特尔的转型决策                           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   📋 决策内容                                           │
│   ─────────────                                         │
│   • 彻底退出内存芯片业务                                │
│   • 全面转向微处理器业务                                │
│   • 将资源集中到更有竞争力的领域                        │
│                                                         │
│   📊 转型结果                                           │
│   ─────────────                                         │
│   • 微处理器业务蓬勃发展                                │
│   • 成为个人电脑时代的核心供应商                        │
│   • 公司市值大幅增长                                    │
│   • 奠定了英特尔在全球半导体行业的领导地位              │
│                                                         │
│   🏆 历史评价                                           │
│   ─────────────                                         │
│   • 这次转型被誉为商业史上最成功的转型之一            │
│   • 展示了"大反思"的力量和效果                        │
│   • 成为商学院经典案例                                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 4.4 案例启示

英特尔案例给我们的启示：

| 启示 | 说明 |
|-----|------|
| **放下过去** | 过去的成功可能成为未来的包袱 |
| **外部视角** | 用"新CEO"视角审视自己，打破思维定势 |
| **质疑根本** | 敢于质疑最核心的业务和假设 |
| **果断行动** | 大反思之后需要果断的行动 |
| **时机关键** | 在危机时刻，大反思可能挽救组织 |

## 五、如何进行大反思

### 5.1 大反思的提问框架

```
┌─────────────────────────────────────────────────────────┐
│              大反思提问清单                             │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   关于基本假设                                          │
│   ─────────────                                         │
│   □ 我们对行业的基本假设还成立吗？                      │
│   □ 我们对客户的理解是否过时了？                        │
│   □ 我们的竞争优势是否还存在？                          │
│   □ 有没有我们从未考虑过的新可能性？                    │
│                                                         │
│   关于目标和方向                                        │
│   ────────────────                                      │
│   □ 如果我们今天从零开始，还会选择同样的方向吗？        │
│   □ 有没有更重要的目标被我们忽视了？                    │
│   □ 我们的目标是否还值得追求？                          │
│   □ 五年后，我们希望成为什么样的组织？                  │
│                                                         │
│   关于心智模式                                          │
│   ─────────────                                         │
│   □ 我们的思维方式是否限制了我们的视野？                │
│   □ 有没有完全不同的商业模式可以参考？                  │
│   □ 如果我们是一家新公司，会怎么做？                    │
│   □ 我们的"不可能"是真的不可能，还是我们以为不可能？  │
│                                                         │
│   关于行动                                              │
│   ─────────                                             │
│   □ 如果必须改变一件事，那应该是什么？                  │
│   □ 什么是我们现在就应该停止做的？                      │
│   □ 什么是我们现在就应该开始做的？                      │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 5.2 大反思的方法

| 方法 | 说明 | 适用场景 |
|-----|------|---------|
| **归零思考** | 假设从零开始，会如何选择 | 业务方向迷茫时 |
| **外部视角** | 用外部顾问/新CEO视角审视 | 陷入思维定势时 |
| **极端假设** | 假设核心业务消失，如何应对 | 过于依赖单一业务时 |
| **跨界对标** | 学习其他行业的做法 | 寻求创新突破时 |
| **逆向思考** | 思考如何失败，然后避免 | 风险评估时 |

## 六、口诀应用：个人与团队

### 6.1 个人层面的应用

> **"我要改变什么？"不仅是一句口诀，更是一种思维方式。**

```
┌─────────────────────────────────────────────────────────┐
│              个人应用指南                               │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   日常反思                                              │
│   ─────────                                             │
│   • 每天问自己：今天我要改变什么？                      │
│   • 每周回顾：这周最大的收获和需要改变的地方            │
│   • 每月深度反思：是否有需要"大反思"的地方            │
│                                                         │
│   关键决策前                                            │
│   ────────────                                          │
│   • 使用"归零思考"：如果重新开始，会怎么选？          │
│   • 使用"外部视角"：如果我是旁观者，会建议什么？      │
│   • 区分小改变和大改变：这是优化还是转型？              │
│                                                         │
│   遇到困境时                                            │
│   ────────────                                          │
│   • 暂停，进行深度反思                                  │
│   • 质疑基本假设：我的前提条件对吗？                    │
│   • 寻找大反思的机会：这是改变的契机                    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 6.2 团队层面的应用

领导者不仅应自己运用"我要改变什么？"这句口诀，还应鼓励团队成员运用，共同推动组织和个人的成长与进步。

```
┌─────────────────────────────────────────────────────────┐
│              团队应用指南                               │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   建立反思文化                                          │
│   ────────────                                          │
│   • 鼓励团队成员提出"我们需要改变什么"                │
│   • 创造安全的反思环境，不惩罚坦诚的反思                │
│   • 将反思纳入团队常规流程（会议、复盘等）              │
│                                                         │
│   引导团队反思                                          │
│   ────────────                                          │
│   • 在团队会议上问："我们的基本假设还成立吗？"        │
│   • 鼓励不同层级的反思（小、中、大）                    │
│   • 当团队陷入细节时，引导进行大反思                    │
│                                                         │
│   组织层面的反思                                        │
│   ────────────────                                      │
│   • 定期组织战略反思会                                  │
│   • 邀请外部视角参与反思                                │
│   • 将反思成果转化为行动计划                            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 7.2 行动清单

**对于个人：**
- [ ] 今天开始，每天问自己一次"我要改变什么？"
- [ ] 识别当前最需要大反思的领域
- [ ] 使用"归零思考"或"外部视角"进行一次深度反思
- [ ] 将反思转化为具体的改变行动计划

**对于团队：**
- [ ] 在下次团队会议上引入"我们要改变什么"的讨论
- [ ] 建立定期的团队反思机制
- [ ] 鼓励团队成员提出需要改变的观察
- [ ] 学习英特尔案例，讨论团队的大反思时机

## 八、思考与反思

### 反思问题

1. **认知层面**：你平时的反思停留在哪个层次？小反思、中反思还是大反思？

2. **实践层面**：你最近一次进行"大反思"是什么时候？关于什么问题？

3. **应用层面**：在你的工作或生活中，有什么领域需要进行"大反思"？

4. **障碍层面**：什么阻碍了你进行大反思？是思维定势、情感依恋还是其他？

5. **行动层面**：你能设计一个"归零思考"的场景，重新审视某个重要决策吗？
