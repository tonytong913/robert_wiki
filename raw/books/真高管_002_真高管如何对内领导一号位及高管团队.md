# 真高管 · 第002页：真高管如何对内领导一号位及高管团队？

## 原文

**左页：**

真高管如何对内领导一号位及高管团队？

首席组织官对"真高管"的定义及衡量标准是：那些在所辖领域，对内可以领导一号位及高管团队，对外可以代表公司竞争力的人。真高管在精神底色上是创业者。

那可能有人就会问，一号位是高管的上级，高管团队的成员彼此之间是平级，这个真高管怎么就能够对内领导一号位和高管团队呢？

我们说的"领导"并不是行政上的汇报关系，并不是简单的"我说了算"。

我们说的"领导"是指，对一号位和高管团队的影响力。比如，能否就方向的设定、目标的设定有影响力？能否引导关键路径和关键任务的选择？

**右页：**

为帮助大家理解，我举两个小例子：

真高管在所辖领域，可以基于自己的行业洞见和专业理解去主动发现一些机会，然后带着方案向一号位和高管团队提出业务增长的可能的方向和实现的路径。即"什么是正确的事"、"如何把事情做正确"，从而实现公司在该领域的对外竞争力。

有时候一号位也会凭直觉大致的给出一个模糊的方向，真高管也可以基于自己过往的经验及专业洞见去理解和把握一号位想表达的意思。他可以用更简单、更全面的语言去复述一号位想表达的东西。在这过程当中，他甚至可以比一号位说得还明白。同时，也可以让其他一头雾水的高管们听得明白。最后，还能够跟庖丁解牛一般，向一号位和高管团队描述下一步的关键任务。通过这些关键任务、里程碑的设定，来帮助一号位和高管团队去锁定想要的方向和目标，从而实现对一号位和高管团队的领导。

---

## 核心概念

| 概念 | 说明 |
|------|------|
| **"领导"=影响力** | 领导不是行政汇报关系，不是"我说了算"，而是对方向、目标、路径的影响力 |
| **主动开荒** | 基于行业洞见和实践理解，主动发现机会并提出方案 |
| **翻译一号位** | 把一号位的模糊直觉转化为清晰、全面的表述，让所有人都明白 |
| **庖丁解牛式引导** | 拆解关键任务和里程碑，帮助团队锁定方向和目标 |
