# 领导力导论01讲：领导力其实是领袖力

## 核心观点

领导力 ≠ 职位，领导力的本质是**责任**——解决问题的责任。

---

## 一、领导力的定义与本质

### 1. 领导力是责任
- 核心不是职位或权力，而是**解决问题的责任**
- 这种责任是一个人无法单独解决，需要带领一群人共同解决的问题

### 2. 领导力与职位的区别
| 误区 | 正解 |
|------|------|
| 领导力 = 职位 | 领导力 ≠ 职位，但两者有关联 |
| | **领导力是领导职位的责任** |
| | **领导职位是领导力的资源** |

> 将领导力等同于职位，是对领导力最大的误解。

---

## 二、领导与领袖的区别

| | 领导 | 领袖 |
|--|------|------|
| **定义** | 担任领导职位的人 | 真正发挥领导力的人 |
| **职位** | 有职位 | 可能有，也可能没有 |
| **共同点** | - | 承担责任并解决问题 |

### 学院派研究的误区
- 学院派研究往往关注"领导"（职位）而非"领袖"（实质）
- 导致研究视角出现偏差

---

## 三、领导力的来源与展现

### 来源
- **承担责任**：特别是解决团体或组织面临的问题

### 展现
- **带领团队**：不是个人能力，而是带领团队共同解决问题、实现目标

---

## 四、课程主题与目的

### 主题
**如何当领袖** —— 如何发挥领导力，而非如何获得领导职位

### 目的
- 通过"十句口诀"等方法
- 培养学员成为**解决问题的高手**
- 从而真正发挥领导力

---

## 五、思考题与讨论

### 课后思考题
**领导力的词性是什么？**
- 名词？动词？形容词？
- 结合领导力的本质进行思考

### 学员观点精选
1. **合体说**：领导力是名词、动词与形容词的合体
2. **动名词说**：强调其行动力和影响他人的能力

---

## 总结

> 本课程旨在培养**发挥领导力的领袖**，而非仅仅追求领导职位的人。

**关键转变**：从关注"职位"转向关注"责任"，从"个人"转向"团队"，从"权力"转向"解决问题"。
