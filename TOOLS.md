# TOOLS.md - 工具配置

## 技能与工具

Skills 定义工具如何工作。这个文件用于记录环境特定的配置。

## Wiki 系统工具

### 目录结构
- `raw/` — 存放原始文档
- `wiki/` — LLM 生成的知识页面
- `memory/` — 记忆文件
- `.learnings/` — 学习记录

### 文件命名规范
- 源文档：`YYYY-MM-DD-来源简称.md`
- 每日记忆：`YYYY-MM-DD.md`

## 可选集成

### Obsidian
- 用于浏览和编辑 Wiki
- Graph View 查看知识图谱
- Dataview 查询元数据

### Obsidian Web Clipper
- 浏览器扩展，快速保存网页文章

### Git
- 版本控制 Wiki 内容
