# GPT Image 2 提示词编写指南

> 来源：OpenAI 官方 Cookbook（2026-04-21）、Atlabs AI Ultimate Guide（2026-04-23）
> 用途：供人类和 AI 参考的 GPT Image 2 提示词最佳实践

---

## 核心公式

GPT Image 2 是自然语言模型，reward **清晰的结构化描述**，而非关键词堆砌。

### 标准公式

```
[主体 + 形容词] [动作] [场景/背景]
[构图/镜头]
[光线/氛围]
[风格/媒介]
[精确文字 + 排版]
[用途/比例]
```

### 示例

```
主体：哑光黑色陶瓷咖啡杯，微浮雕纹理
动作：放在湿石板台面上，旁边折叠亚麻餐巾
场景：极简北欧厨房，日出时分
构图：3/4 角度，50mm 镜头，浅景深
光线：柔和侧窗光，清晨冷调，轻微边缘高光
风格：编辑级产品摄影，自然胶片颗粒
文字：杯身印有 MONDAY，细体大写无衬线字，清晰可读
```

## 十大核心原则

### 1. 结构化 + 明确目标

按固定顺序写提示词（场景 → 主体 → 细节 → 约束），并标注用途（广告/UI 原型/信息图）以设定"模式"和精美程度。

```
错误：做一个咖啡机的图
正确：创建一张自动咖啡机工作原理的信息图。从豆仓→研磨→秤→水箱→锅炉，技术性+视觉化呈现。9:16 竖版，矢量风格。
```

### 2. 具体且具体

- **材质**：哑光 vs 光泽、木质 vs 金属、棉麻 vs 丝绸
- **视觉媒介**：照片、水彩、3D 渲染、线稿
- **质感关键词**：胶片颗粒、纹理笔触、微距细节
- **写实关键词**：直接使用 `photorealistic`、"真实照片"、"iPhone 拍摄" 等词能强烈激发写实模式

### 3. 不用关键词堆砌

GPT Image 2 不需要 Midjourney 风格的 `8k, masterpiece, trending on artstation`。它读得懂自然语言。

```
错误：8k masterpiece photorealistic coffee minimalist trending
正确：极简风格产品摄影，哑光白陶瓷咖啡杯，自然窗光，浅景深
```

### 4. 构图控制

| 元素 | 示例 |
|------|------|
| 景别 | 特写、中景、全景、俯拍 |
| 角度 | 平视、仰角、45度 |
| 镜头 | 50mm、85mm、广角、微距 |
| 光线 | 柔和散射、金色时刻、高对比度、霓虹 |
| 布局 | "logo 右上角""主体居中，左侧留白" |

### 5. 人物控制

描述**比例、取景框、视线、互动**：

```
全身可见，包含双脚
相对于桌子是儿童身高
低头看书，不看镜头
双手自然握着手把
```

这些细节直接影响身体比例、动作几何和视线对齐。

### 6. 约束（改什么 vs 保留什么）

**生成时**显式排除：
```
无水印、无额外文字、无 logo/商标、无边框
```

**编辑时**精确指定：
```
只改 X，其他一切保持不变
保留：身份/几何/布局/品牌元素/饱和度/对比度
```

### 7. 图片内文字

```
用引号或全大写标注文字内容："SUMMER COLLECTION"
指定排版：字体风格、大小、颜色、位置
生僻词逐字母拼写：品牌名 "Q-U-A-N-T-A-R-I-U-S" 
小字或密集文字用 quality=high
```

### 8. 多图输入

```
用索引+描述引用每张图：
"图1：产品照片… 图2：风格参考…"
描述交互方式：
"将图2的风格应用到图1"
"把图1的鸟放到图2的大象上"
```

### 9. 迭代而非一次塞满

长提示可以工作，但调试困难：

```
1. 先写干净的基础提示词，生成初版
2. 用小的单点修改迭代：光线暖一点 → 去掉多余的树 → 恢复原背景
3. 改背景时检查是否保留了：主体身份、光照方向、饱和度、原有文字
```

### 10. 质量 vs 速度

| quality | 适用场景 |
|---------|---------|
| `low` | 高吞吐、快速实验、草稿预览（多数场景已足够） |
| `medium` | 一般生产环境 |
| `high` | 密集文字、信息图、人像特写、身份敏感编辑 |

## 十大实用模板

### 1. 产品广告
```
创建 [产品] 的干净产品广告。柔和中性背景的工作室灯光，微妙投影，居中构图。左上角现代无衬线标语：'[标语]'。高画质模式。
```

### 2. 电商带文字
```
[产品] 的高端产品照，纹理表面上。自然窗光，写实材质。正面标签精确文字：'[品牌名]'位于上三分之一，'[型号]'位于下方。无多余字，完美字间距衬线体。
```

### 3. 细节特写人像
```
[主体] 的极限特写人像。放大 [具体细节]。浅景深，85mm f/1.4 感，大窗柔光。写实皮肤纹理，无美颜感。
```

### 4. 多参考图合成
```
用附带参考图生成新场景：图1的精确角色 + 图2的服装 + 图3的姿势 + [新环境]。保留：面部、发型、面料纹理、品牌 logo 位置。电影光，4K。
```

### 5. 品牌指南
```
用附带照片为灵感，生成 [业务类型] 的完整品牌指南页。包含：logo（主+单色）、色板+色号、字体搭配、示例社媒帖、包装模型。单页清晰布局，可直接生产使用。
```

### 6. 信息图
```
生成 [主题] 的分步信息图。使用真实准确的信息。简洁矢量风格，柔和色板，编号步骤配图标和简短标注。顶部标题：'[标题]'。9:16 竖版。
```

### 7. 杂志拼贴
```
创作 [主题] 的杂志编辑页面。混合媒材拼贴感：4-6 张不同尺寸照片，撕纸边缘，手写批注，一个醒目标题：'[标题]'。印刷杂志美学，轻微纹理纸张背景。
```

### 8. App 截图
```
生成 [App/平台] 对话的写实截图。包含标准 UI（时间、电量、导航栏）。精确展示：[粘贴消息]。字体和间距匹配真实 App。无水印。
```

### 9. 电影级宽幅概念
```
21:9 超宽电影概念艺术：[场景描述]。体积光，大气雾霾，遮罩绘画风格，前景超细节+背景柔焦。史诗尺度，Syd Mead 风+Roger Deakins 风。
```

### 10. JSON 结构化人像
```
按以下规格生成写实人像：{
  "主体": "30岁女性，淡妆，直视镜头",
  "服装": "oversized 奶油色针织毛衣",
  "场景": "极简公寓，午后柔光从左侧来",
  "镜头": "50mm, f/2.0, 平视, 3/4 身取景",
  "情绪": "平静、抓拍感、不做作",
  "调色": "柔和自然色调，轻微胶片颗粒"
} 严格遵循规格。不加额外道具或文字。
```

## 常见错误

| 错误 | 正确做法 |
|------|---------|
| 关键词堆砌 `8k masterpiece` | 用自然语言描述画质和风格 |
| "加个标题"（太模糊） | `标题文字："SUMMER COLLECTION"，粗体大写衬线，居中，白字炭灰底` |
| 默认 4K（实验性，不稳定） | 先 quality=low 生成，再用专用放大工具 |
| 编辑时忘记 preserve 列表 | 每次迭代重复 `保留：身份/光照/布局/原文字` |

## 提示词模板（供 AI 参考调用）

当用户要求生成图片时，可按以下结构组织提示词：

```
任务：[产品广告 / 信息图 / 概念艺术 / 人像 / UI 模型]
主体：[品类 + 核心特征 + 材质]
场景：[环境 + 背景]
构图：[景别 + 角度 + 镜头焦段]
光线：[类型 + 方向 + 色温]
风格：[摄影/3D/插画/矢量 + 具体风格参考]
文字：[精确内容 + 字体 + 位置]（如有）
约束：[排除项 + 保留项]
比例：[1:1 / 9:16 / 16:9 / 21:9]
```
